# Regex Dictionary Enhanced
A dictionary searchable with regex, with the ability to select between various different dictionaries

Made because all other regex dictionaries don't offer:
- selecting which dictionary to use
- selecting which regex engine to use (TODO)
- dark mode
- definitions

### Wordlist Sources
- [General](https://github.com/dwyl/english-words)
- [Scrabble CSW24](https://drive.google.com/file/d/1Gg7E_lqho9IUA1koBb_xAMo_lHIczVWN/) [(thanks random redditor!)](https://www.reddit.com/r/scrabble/comments/1obxln2/comment/nkk8zv3/)
